三、毕设过程详细说明
本毕业设计基于YOLOv8深度学习模型，实现PCB电路板缺陷检测系统。系统整体流程包括数据预处理、模型训练、目标检测、结果可视化以及GUI界面展示。
在开发过程中，首先完成了数据集整理与标注工作，然后搭建YOLOv8训练环境进行模型训练。随后基于PyQt5开发了可视化界面，实现图片检测、视频检测及摄像头实时检测功能，使系统具有较好的交互性与实用性。

四、遇到的问题及解决方法
1. 数据集缺失问题
在项目初期，存在数据集样本不足的问题，导致模型训练效果不理想。
解决方法：
通过数据增强方式扩充数据集，包括图像翻转、旋转、缩放、亮度调整等方法，提高数据多样性，从而提升模型泛化能力。

2. 项目结构不清晰
在开发过程中，代码文件较为分散，结构不够规范，影响后期维护与扩展。
解决方法：
重新对项目结构进行了整理，将代码按功能模块划分，如模型训练模块、GUI界面模块、工具函数模块等，使整体结构更加清晰规范，便于后续维护与论文整理。


 五、实践总结

通过本周的实践与开发工作，我对基于深度学习的目标检测系统有了更加深入的理解。在毕业设计的推进过程中，不仅巩固了YOLOv8模型的使用方法，也提升了实际工程开发能力，包括数据处理、模型训练、系统集成以及界面设计等多个方面。

总体而言，本周的实践不仅推动了毕业设计的进一步完善，也提升了我的综合实践能力和工程开发能力，为后续论文完成和毕业答辩打下了坚实基础。
